Design Web

Um pouco sobre a matéria:

O web designer projeta sites, blogs, sistemas, aplicativos para web e peças digitais como banners e artes para redes sociais.

Em design web, estudamos a linguagem de marcação chamada HTML. Ela permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.
